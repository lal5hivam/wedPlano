# Deployment Checklist

## Pre-Deployment

### Backend Preparation
- [ ] All dependencies installed (`npm install` in backend/)
- [ ] `.env` file configured locally for testing
- [ ] Firebase service account JSON downloaded
- [ ] Cloudinary account created and credentials ready
- [ ] JWT secret generated (minimum 32 characters)
- [ ] Local backend tested (`npm run dev`)

### Frontend Preparation
- [ ] All dependencies installed (`npm install` in frontend/)
- [ ] Local frontend tested (`npm run dev`)
- [ ] Build tested locally (`npm run build`)
- [ ] API integration tested with local backend

## Backend Deployment (Railway)

- [ ] Railway CLI installed (`npm install -g @railway/cli`)
- [ ] Logged into Railway (`railway login`)
- [ ] Project initialized (`railway init`)
- [ ] Environment variables set:
  - [ ] NODE_ENV=production
  - [ ] JWT_SECRET
  - [ ] JWT_EXPIRES_IN
  - [ ] FIREBASE_PROJECT_ID
  - [ ] FIREBASE_PRIVATE_KEY_ID
  - [ ] FIREBASE_PRIVATE_KEY
  - [ ] FIREBASE_CLIENT_EMAIL
  - [ ] FIREBASE_CLIENT_ID
  - [ ] FIREBASE_AUTH_URI
  - [ ] FIREBASE_TOKEN_URI
  - [ ] CLOUDINARY_CLOUD_NAME
  - [ ] CLOUDINARY_API_KEY
  - [ ] CLOUDINARY_API_SECRET
  - [ ] FRONTEND_URL (temporary, update after Netlify)
- [ ] Backend deployed (`railway up`)
- [ ] Railway domain obtained (`railway domain`)
- [ ] Health endpoint tested (`/health`)
- [ ] API docs accessible (`/api/docs`)

## Frontend Deployment (Netlify)

- [ ] Netlify CLI installed (`npm install -g netlify-cli`)
- [ ] Logged into Netlify (`netlify login`)
- [ ] `.env.production` updated with Railway backend URL
- [ ] Frontend built (`npm run build`)
- [ ] Deployed to Netlify (`netlify deploy --prod`)
- [ ] Environment variable set:
  - [ ] VITE_API_URL (Railway backend URL)
- [ ] Netlify URL obtained
- [ ] Site accessible and loads correctly

## Post-Deployment Configuration

### Backend Updates
- [ ] FRONTEND_URL updated in Railway with Netlify URL
- [ ] Backend redeployed with new CORS settings

### Firebase Configuration
- [ ] Netlify domain added to Firebase authorized domains
- [ ] Railway domain added to Firebase authorized domains
- [ ] Authentication tested from deployed frontend

### Cloudinary Configuration
- [ ] Netlify domain added to Cloudinary allowed domains
- [ ] Railway domain added to Cloudinary allowed domains
- [ ] Image upload tested from deployed frontend
